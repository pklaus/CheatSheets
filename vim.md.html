VIM Text Editor

The three modes of vi

  • Command Mode
  • Insert Mode
  • Last Line Mode

Editing

i insert o enter a new line above O enter a new line below

Cutting and Pasting

x delete current character 8x delete 8 characters to the right dd delete the current line 4dd delete the current plus 3 more lines
